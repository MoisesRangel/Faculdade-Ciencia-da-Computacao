# Faculdade — Ciência da Computação

Repositório dedicado aos resumos, exercícios e atividades desenvolvidos durante minha graduação em Ciência da Computação (Estácio).

## 📂 Estrutura

O repositório é organizado por **disciplina**, e dentro de cada disciplina por **Tema/Módulo**, seguindo a divisão de conteúdo da própria faculdade.

```
├── Computacao-em-Nuvem/
│   └── Tema-N-[assunto]/
│       ├── resumo-temaN-[assunto].md
│       ├── audio-resumo-temaN-[assunto].m4a
│       └── audios/
├── Fundamentos-de-Redes-de-Computadores/
├── Introducao-a-Seguranca-da-Informacao/
├── Matematica-e-Logica/
└── Introducao-a-Programacao-em-Computadores/
    └── exercicios/
```

Dentro de cada pasta de Tema:

| Item | O que é | Público? |
|---|---|---|
| `resumo-temaN-[assunto].md` | Resumo de aula escrito por mim, com tópicos, tabelas e analogias | ✅ Sim |
| `audio-resumo-temaN-[assunto].m4a` | Áudio-resumo gerado pelo NotebookLM a partir do meu material de estudo | ✅ Sim (via Git LFS) |
| `audios/` | Gravações de aula feitas pelo professor | 🔒 Não (fora do controle de versão) |

- Nem todo Tema é documentado — o Tema 1, geralmente introdutório, costuma ser registrado apenas quando traz conteúdo relevante.
- Áudios de aula gravados pelos professores **não são versionados neste repositório**, por se tratar de conteúdo de propriedade da instituição/professor. Eles ficam apenas em ambiente local, dentro da pasta `audios/`, que é ignorada via `.gitignore`.
- Arquivos de áudio públicos (`.m4a`, `.mp3`, `.wav`) são versionados via **Git LFS**, para manter o repositório leve.

## 🎯 Objetivo

Este repositório serve tanto como **registro de estudo** quanto como parte do meu portfólio em construção, documentando minha trajetória e evolução ao longo da graduação.
